## Overview

Welcome back! Veteris is back online with a significantly expanded catalog of classic iOS apps!

## Installation

Add `https://repo.victorlobe.me` to Cydia and install **Veteris**.

The project website is available at `https://veteris.victorlobe.me`.

## Requirements

- Jailbroken iPhone, iPad, or iPod touch
- iOS 5.0 or newer
- AppSync
- Veteris Helper, installed automatically as a dependency
